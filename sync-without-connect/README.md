## 機能
* 「更新(同期)」ボタンを押し、「再レンダリング」ボタンを押すと、ハードコーディングされた文言が描画されます。
* 「リセット」ボタンを押し、「再レンダリング」ボタンを押すと、文言が初期化されます。
